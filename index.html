Introducción

Este proyecto implementa un compilador completo con análisis léxico, sintáctico y semántico, incluyendo una extensión para NPCs.

Lenguaje

x = 10
si x > 5:
    escribir(x)

Fase Léxica

El código se convierte en tokens.

Fase Sintáctica

Se construye el AST.

Fase Semántica

Se valida la lógica del programa.

Lenguaje NPC

npc guardia:
    hablar("Guardia", "Alto")
